GIT
git init-> crea repositosio local
git add . --> agrega todos los cambios  
git add archivo.txt-->agrega cambios especificos
git status--> visualiza cambios(rojo pendiene, verde guadados)
ctrl+L -->Limpia pantalla
git commit -m "Mensaje de los cambios"
git log--> fecha y hora de commit
git push origin master-->subir los cambios al origen
git pull origin master-->bajar los cambios del origen
git clone ruta de la web(code)--> clonarlo que esta en la nube al local(mi pc)
cd MiSegundoRepositorio/--> para entrar al epositorio despues de clonar
